Leaks In The Pipe

An interactive storytelling project on Black Women in STEM.

Main Ingredients

  • HTML
  • CSS
  • JavaScript
  • Svelte
  • Intersection Observer API

Chef's Note

Using real-world data from Black Wealth Data Center, I created a web-based experience that explores the representation of Black Women in STEM industries, learning new technologies like Svelte and the Intersection Observer API along the way.
